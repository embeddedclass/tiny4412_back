Build NFS environment

1. sudo apt-get install nfs-kernel-server nfs-common portmap

2. configure portmap
   1.	sudo vim /etc/default/portmap
   2.	del 127.0.0.1

3. comfigure exports
   1.	sudo gedit /etc/exports
   2.	add ur share file forder

   example :
 	 /home/cadtc/tiny4412/Day1/root_mkfs *(rw,sync,no_root_squash)

   	*: any ip can entry this

4. restart 

   1.	sudo /etc/init.d/portmap restart
   2.	sudo /etc/init.d/nfs-kernel-server restart 

5. Host :
	ifconfig ethx 192.168.0.10

6.	sudo tar -xvjf root_mkfs.tar.bz2


7. local test
   1.	sudo mount -t nfs 192.168.0.10:/home/cadtc/tiny4412/Day1/root_mkfs /mnt/

8.	ls -l /mnt

9.	sudo umount /mnt
